Final Report

Project Summary

This project successfully demonstrated the integration of Knowledge Graphs and Large Language Models for cultural heritage enrichment. Through systematic SPARQL queries and LLM-based knowledge extraction, we created 50+ new RDF triples that enhance the representation of Palmyra's post-conflict heritage status.

Key Achievements

  • Identified 4 critical gaps in existing knowledge graph representations
  • Created custom vocabulary for heritage authenticity and restoration
  • Generated 50+ enriched RDF triples with verified information
  • Compared three major LLMs for heritage knowledge extraction
  • Demonstrated "Composite Authenticity" framework for post-conflict sites

Recommendations

  • Integrate enriched data into Wikidata for broader accessibility
  • Establish community-driven verification processes for LLM outputs
  • Develop standardized vocabularies for post-conflict heritage sites
  • Create automated pipelines for continuous knowledge graph enrichment
