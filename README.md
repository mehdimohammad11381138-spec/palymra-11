# Palmyra Heritage Project (Simple HTML)

This is a **framework-free** static website version of your project, using:

- `index.html`
- `styles.css`
- `script.js`

## Publish on GitHub Pages

1. Create a new GitHub repository (or use an existing one).
2. Upload these files to the repository root.
3. In GitHub: **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: **main** (or `master`) and folder: **/ (root)**
4. Save. Your site will be available at the Pages URL GitHub shows.

## Tabs & deep links

Tabs are controlled by the URL hash, e.g.

- `#home`
- `#context`
- `#methodology`
- `#sparql-queries`
- ...

You can link directly to a tab from anywhere.
